### 1. 고가용성과 지연-형평성을 고려한 Ticketing System

> **국가별 시차와 네트워크 지연을 극복한 공정한 티켓팅 서비스**

*   **배경 및 동기**: 기존 시스템은 네트워크 지연 차이를 해소하고자 쿼터제를 운영했으나, IP 우회 및 위장 가입 등 보안 이슈가 발생함. RTT를 고려하지 않는 불공정성 문제를 해결하고자 함.
*   **주요 목표**: 
    *   제한(쿼터제) 없는 공정한 티켓팅 서비스 제안함.
    *   Region to DB의 RTT와 쿼리 타임스탬프를 계산하여 임의 지연을 부여하는 형평성 시스템 구축함.
    *   ALB, ASG 기반의 고가용성 확보 및 이상 접속 차단함.
*   **결과물**: 지역 제약 없이 동시성을 갖춘 공정한 티켓팅 서비스 아키텍처임.
*   **기대 효과**: 제한 없는 서비스 제공 및 글로벌 서비스 확장에 적용 가능함.
*   **팀원 역할**: 
    *   **지규인**: 시스템 설계, 인프라 구축(ALB, ASG, SG), Frontend(React) 및 핵심 모듈 개발, 부하 테스트 수행함.
    *   **박민석**: 테스트 시나리오 설계, 멀티 리전 EC2/VPC 환경 구축, Backend API/DB 개발, RTT 테스트 수행함.

### 2. 비지도 학습 기반 API 이상 탐지 시스템

> **실시간 탐지와 알림을 통한 서버 보안 강화 서비스**

*   **배경 및 동기**: 24년 침해 사고 신고 건수가 급증했으며, 해킹 경유지 악용 등 서버 해킹 사례가 크게 증가함.
*   **주요 목표**:
    *   Isolation Forest 모델을 활용한 레이블링 없는 비지도 학습 기반 이상 탐지 구현함.
    *   API 요청 즉시 분석 및 Slack을 통한 실시간 알림 시스템 구축함.
    *   Lambda 기반의 비용 효율적인 클라우드 아키텍처 설계함.
*   **결과물**: AWS Learner Lab 제약을 극복하고 높은 정확도를 달성한 실시간 탐지 시스템임.
*   **기대 효과**: 새로운 공격 패턴의 자동 학습 및 보안 특징 엔지니어링을 통한 방어 능력 향상됨.
*   **팀원 역할**: 
    *   **신창희, 허진**: 프로젝트 개요 수립, 기술 스택 선정, ML 모델 및 보안 특징 엔지니어링, 실험 결과 분석 수행함.

### 3. 차량 침수 재해 실시간 경보 서비스

> **실시간 센서 데이터와 위치 정보를 결합한 차량 침수 피해 예방 시스템**

*   **배경 및 동기**: 여름 침수 차량이 23년 기준 약 2만 대를 상회하며 막대한 경제적 피해가 발생함. 실시간 위치 기반의 맞춤형 경보 체계가 필요함.
*   **주요 목표**:
    *   FloodnetNYC 오픈 데이터를 활용한 실시간 홍수 모니터링 및 시뮬레이션 구축함.
    *   차량 위치 기준 최근접 센서를 조회하여 4단계 침수 경보 시스템 적용함.
    *   침수 구역을 회피하여 안전 센서와 고지대로 이어지는 3단계 대피 경로 제공함.
*   **결과물**: React 기반 인터페이스와 FastAPI 서버를 연동한 실시간 침수 경보 및 대피 안내 시스템임.
*   **기대 효과**: 경제적 피해 감소 및 실시간 위치 기반 맞춤 경보를 통한 인명·재산 보호됨.
*   **팀원 역할**: 
    *   **진성원, 박진영**: 시스템 아키텍처 설계, GPS 시뮬레이션 및 다중 차량 관리 기능 구현, 대피 경로 알고리즘 개발함.

### 4. Cloud-based Automatic Sunvisor System (CASS)

> **태양 위치 계산을 통한 운전자 눈부심 예측 및 자동 대응 시스템**

*   **배경 및 동기**: 운전 중 눈부심은 사고 위험 요인이지만 실시간 대처가 어려우며, 기존 AI 눈 추적 방식은 프라이버시 및 비용 문제가 존재함.
*   **주요 목표**:
    *   차량 GPS, 방향, 시간을 기반으로 NOAA 알고리즘을 사용해 태양 위치(방위각, 고각) 계산함.
    *   출발·도착점 경로 상의 눈부심 구간을 예측하고 선바이저 전개 방향 설정함.
    *   클라우드를 통한 데이터 저장으로 계산 비용 절감 및 동일 경로 차량 간 정보 공유함.
*   **결과물**: AWS IoT Core 및 Lambda를 활용한 서버리스 기반 자동 선바이저 권고 시스템임.
*   **기대 효과**: 자율주행 안전성 향상 및 계산 비용 최적화된 사용자 맞춤 서비스 제공됨.
*   **팀원 역할**: 
    *   **정성윤, 최한슬**: 시스템 구조 및 알고리즘 설계, AWS 클라우드 서비스 연동, DynamoDB 활용 이력 저장 구현함.
*   [결과물](materials/4.pdf)

### 5. Spike-driven Privacy-Preserving Object Detection

> **SNN을 활용한 프라이버시 보존형 고효율 객체 탐지 프레임워크**

*   **배경 및 동기**: 자율주행 작업 중 데이터/모델 노출 등 개인정보 이슈와 고성능 연산 자원에 따른 전력 소모 부담 해결이 필요함.
*   **주요 목표**:
    *   보안성(SecureV2X), 실시간성(SNN), 확장성(Cloud)을 동시에 만족하는 프레임워크 제안함.
    *   YOLOv5의 Backbone을 Spiking Neural Network(SNN)로 치환하여 저전력·저지연 도모함.
    *   2-PC(Multi-Party Computation) 환경에서 암호화된 데이터를 처리함.
*   **결과물**: 기존 ANN용 보안 프레임워크와 SNN 결합 시 발생하는 기술적 한계 및 구조적 미스매치 분석 보고서임.
*   **기대 효과**: 향후 SpikeYolo 등 완전 SNN 기반 모델의 2-PC 통합을 위한 연구 방향성 제시함.
*   **팀원 역할**: 
    *   **우정흠**: SNN 기반 객체 탐지 실험 설계, ANN-to-SNN 변환 시도 및 결과 분석 수행함.
*   [결과물](https://github.com/realbb/PBL25_Spike-Driven-Privacy-Preserving-Yolo)

### 6. LLM 기반 이상 탐지 및 AWS 보고서 생성 서비스

> **AWS 사용량의 지능형 분석 및 사용자 맞춤형 리포트 자동화**

*   **배경 및 동기**: 기존 AWS 사용량 분석은 정적이며 수동 분석에 많은 시간과 지식이 요구됨.
*   **주요 목표**:
    *   Cost Explorer API와 Boto3를 활용해 정기적으로 비용 데이터를 수집 및 필터링함.
    *   Gemma LLM을 EC2에서 구동하여 비용 요약, 원인 분석, 조치 권고를 포함한 자연어 리포트 생성함.
    *   이메일(Smtplib)을 통한 다국어 PDF 리포트 발송 및 CloudWatch 모니터링 체계 구축함.
*   **결과물**: 매일 자정 자동 실행되어 사용자에게 통찰력을 제공하는 클라우드 비용 분석 파이프라인임.
*   **기대 효과**: 개발자·운영자의 분석 시간 소모 절감 및 지능형 비용 최적화 방안 제시함.
*   **팀원 역할**: 
    *   **신성환**: 데이터 수집, 파이프라인 설계, EventBridge & Lambda 자동화, 보안 관리함.
    *   **이현빈**: 데이터 전처리, LLM 프롬프트 구성 및 리포트 생성, 이메일 발송 시스템 구현함.
* [결과물](https://github.com/HBL03/ConnectedCarPBL)

### 7. 기밀 컴퓨팅 기반 딥페이크 음성 탐지 서비스

> **confiential computing과 서버리스를 결합한 프라이버시 보호 탐지 엔진**

*   **배경 및 동기**: 음성 데이터 보호가 필수적이며, 탐지 모델 실행 시 발생하는 비용 최적화가 요구됨.
*   **주요 목표**:
    *   SOTA 모델인 AASIST를 활용해 원시 파형 기반의 고성능 스푸핑 탐지 재현함.
    *   AWS Nitro와 Enclave를 활용한 기밀 컴퓨팅 환경에서 데이터를 보호함.
    *   WebSocket API와 Lambda, SQS를 연동하여 EC2 미가동 시에도 요청을 처리하는 서버리스 아키텍처 구축함.
*   **결과물**: 음성 스푸핑 여부를 판별하는 보안성이 강화된 실시간 추론 파이프라인임.
*   **기대 효과**: 개인정보(음성) 노출 방지 및 서버리스 도입을 통한 운영 비용 절감됨.
*   **팀원 역할**: 
    *   **장선우, 조치현**: AASIST 모델 구현, 데이터셋 기반 추론 엔진 구성, AWS 기밀 컴퓨팅 및 서버리스 환경 배포 수행함.

### 8. SNN 기반 GPS Spoofing 탐지 및 클라우드 배포

> **TS-Former 뉴런 모델을 활용한 고효율·저지연 위치 보안 서비스**

*   **배경 및 동기**: 커넥티드카의 GPS 의존성이 심화됨에 따라 위조 신호를 통한 제어권 탈취 위험이 증대됨.
*   **주요 목표**:
    *   인간 신경세포를 모사한 TS-LIF 뉴런과 iSpikformer를 결합한 초경량 TS-Former 모델 개발함.
    *   NVIDIA Jetson 엣지 디바이스에서 실시간성 및 전력 효율성을 검증함.
    *   MSA 기반 아키텍처를 수립하여 OS 비종속적인 클라우드 네이티브 탐지 서비스 구축함.
*   **결과물**: 98.41%의 정확도와 초저지연(샘플당 1.8ms) 성능을 확보한 GPS 보안 시스템임.
*   **기대 효과**: 자율주행 기술의 신뢰성 향상 및 임베디드 환경 온보드 배포 가능성 확보됨.
*   **팀원 역할**: 
    *   **박재현**: TS-Former 모델 구축 및 학습, Flask 기반 추론 서버 구축함.
    *   **박지호**: AV-GPS 데이터셋 분석 및 전처리, 웹 서버 인터페이스 구축함.
*   [결과물](materials/8.pdf)
